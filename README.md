# AI-Agent-Project-OS
Project Creation OS For working with AI Agents.


---

# How to Install

### Step 1

```
git clone https://github.com/VisualDigitalAgency/AI-Project-Generator.git
```

### Step 2

```
sed -i 's/\r$//' generate_ai_project.sh

```
### Step 3

```
chmod +x generate_ai_project.sh
./generate_ai_project.sh my-ai-project murali/my-ai-project

```
